## About TurningCode

Pembuatan Aplikasi Penjadwalan mengusung tema Gamified Learning yang dimana didalam aplikasi tersebut tersedia materi pembelajaran yang mengusung konsep Microlearning (materi singkat,ringan,dan konsisten setiap hari).Dengan mengunakan visual kartun sehingga pembelajaran terasa tidak seperti kelas formal.Serta aplikasi berbasis progres harian yang dimana pengguna diharapkan termotivasi lewat target harian dan pencapaian di aplikasi tersebut.
